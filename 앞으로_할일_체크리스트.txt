카카오톡 대화 웹게시(비밀번호) - 앞으로 내가 할 일 체크리스트

0) 준비물
- 인터넷/브라우저
- (선택) 내 PC에 Python 3.11+ 설치 (비밀번호 해시 생성/로컬 테스트용)

1) 외부서버(PC 없이)로 상시 운영하기: Render (추천)
1-1. 배포 시작(원클릭 링크)
- https://render.com/deploy?repo=https://github.com/ssungjun83/cutesoyeon

1-2. Render에서 반드시 입력할 것(환경변수 1개)
- CHAT_APP_PASSWORD_HASH (비밀번호 “평문”이 아니라 “해시” 값)

해시 만드는 방법(둘 중 하나만)
A) 로컬에서 setup 후 .env에서 복사 (추천)
  1) PowerShell에서 이 폴더에서: .\setup.ps1
  2) 생성된 .env 파일에서 CHAT_APP_PASSWORD_HASH=... 줄의 값을 그대로 복사
B) 로컬에서 해시만 출력해서 복사
  1) .\setup.ps1 (한 번은 필요)
  2) .\.venv\Scripts\python tools\hash_password.py 실행 → 출력값 복사

1-3. 배포 완료 후
- Render가 만들어준 URL로 접속 → 비밀번호 입력 → 상단 “가져오기”에서 카카오톡 .txt 업로드
- 이후에도 새 .txt를 계속 업로드하면 누적 저장됨

중복 제거 기준
- “날짜/시간(분 단위) + 메시지 내용”이 같으면 이름이 달라도 자동으로 중복 제외됨

주의(중요)
- GitHub에는 개인정보 보호를 위해 KakaoTalk_*.txt / 사진 파일(.jpg 등)이 자동으로 업로드 제외되어 있음
- 서버에 대화를 올리려면 “가져오기”에서 직접 업로드해야 함

2) (선택) 내 PC에서 로컬로 실행/확인하기
2-1. 처음 1번
- .\setup.ps1 실행 (가상환경 설치 + 비밀번호 설정)
2-2. 실행
- .\run.ps1 실행
- 브라우저: http://127.0.0.1:8000 접속

3) (선택) 내 PC에서 임시로 인터넷 공유하기(PC 켜져 있어야 함)
- .\share.ps1 실행
- 출력되는 https://....trycloudflare.com 주소를 공유
